## Fases a realizar

### **Fase 1:** 
Identifica los 3 grupos de interés clave y los 3 temas materiales (ej. eficiencia energética del código, accesibilidad universal, privacidad de datos) para una consultora web.

Tenemos 3 grupos de interes (Stakeholders): Clientes empresariales, empleados y los usuarios finales.

**Eficiencia energetica (ambiental):** El consumo energético de los centros de datos es muy poco comparado a todo el consumo energético mundial. Pero en la actualidad se ha intentado optimizar los codigos y el hardware para un menor consumo.
**Accesibilidad universal (social):** Esto se utiliza para garantizar que cualquier persona (con y sin discapcidad) pueda acceder a internet y visitar webs sin ningun tipo de problema.
**Privacidad de datos (gobernanza):** Esto se encarga de proteger la información de los usuarios y cumplir con la ley de proteccion de datos.

Se pide crear un XML llamado materialidad que este bien formado y comentado.
